#smart energy monitoring system 
## OVER VIEW
   this project is an iot based smart energy monitoring system that measures electricity usage and helps in optimizing    power consumption.
##****FEATURES**
-- Real time energy monitoring
-- power calculation (voltage*current)
-- helps reduce electricity wastage
##**TECHNOLOGIES USED**
-- arduino(c++ based programming)
-- current sensor
-- voltage measurement 
-- display 
## HARDWARE COMPONENTS
-- arduino board(uno)
-- current sensor(acs712)
-- voltage sensor
-- connecting wires 
-- bread boad
**##WORKING PRINCIPLE**
the current sensor measures the flow of current flow the load. the arduino reads sensor values and calculation power using the formula
             power=voltage*current

the calculated values are displayed using serial monitoring or display module
**##APPLICATIONS**
-- Home energy monitoring
-- industrial power tracking
-- energy saving system
-- smart iot based solutions
